## Javascript

### .includes()
```js
// Array.prototype.includes()
// arr.includes(valueToFind[, fromIndex])

array.includes(2);
array.includes("cat");
```
배열이 특정 요소를 포함하고 있는지 확인해주는 메서드.
- 반환값: 성공시 true, 실패시 false(boolean)


### .sort()
```js
arr.sort((a, b) => {
  return a - b;
});

arr.sort((a, b) => a - b);
```
중괄호를 써주는 경우엔 `return`을 써줘야 하지만, 중괄호 없이 사용하는 경우엔 `return`을 써주지 않아도 괜찮다.
